# ResumeAI

ResumeAI is an AI-powered resume analysis platform designed to help job seekers improve their resumes and increase their chances of landing interviews.

The application analyzes uploaded resumes, evaluates ATS compatibility, identifies missing keywords and skills, and provides actionable recommendations for improvement.

## Features

* Resume Upload & Analysis
* ATS Compatibility Scoring
* AI-Powered Feedback
* Skill & Keyword Detection
* Job Description Matching
* Resume Performance Dashboard
* Modern SaaS Interface

## Tech Stack

* HTML, CSS, JavaScript
* React (planned)
* Node.js & Express (planned)
* PostgreSQL (planned)
* OpenAI API
* LangChain
